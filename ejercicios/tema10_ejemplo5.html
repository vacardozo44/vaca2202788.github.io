Youtube video ID
Youtube mostrará un iD (como EF81Pb5IWoU), al guardar (o reproducir) un video. Puede utilizar este ID y hacer referencia a su video en el código HTML.

Reproducción de un video de YouTube en HTML
Para reproducir el video en una página web, haga lo siguiente:

  • Sube el video a Youtube
  • Anote el ID del video
  • Defina un elemento <iframe> en una página web.
  • Deje que el atributo src apunte a la URL del video.
  • Utilice los atributos width y height para especificar la dimensión del reproductor.
