Valores RGB Código Hexadecimal Nombre de los Colores
Los valores de rojo, verde y azul se expresan como números entre 0 y 255 Los valores hexadecimales representan valores para rojo, verde y azul en código hexadecimal. Los colores se representan mediante nombres predefinidos. Sin Embargo, son muy limitados en número.
Celda rgb Celda Hex Celda Name
rgb (102, 205.170) #66cdaa MediumAquaMarine
Este color se compone de los siguiente valores:
102 rojo
205 verde
170 azul		 El valor del rojo, 102, se expresa como 66 wn código hexadecimal.
el 205 del verde se expresa como cd
y el 17 del azul equivale a aa.		 Hay 147 nombres de color soportados por los navegadores (este color es MediumAquaMarine). La mayoria considera que esto es una paleta de colores limitados, y es dificil recordar el nombre de cada uno de los colores.
